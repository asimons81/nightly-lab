# Nightly Lab

A batch of small, useful apps/tools built overnight.

## Apps
- **Reply Goal Tracker** (`apps/reply-goal-tracker`) – track daily reply target, streaks, notes, export.
- **Sprint Focus Timer** (`apps/sprint-focus`) – simple work sprints with log + stats.

## Tools
- **Clip Namer** (`tools/clip-namer`) – renames screen recordings with clean timestamps/prefix.

Each app runs locally by opening `index.html` in a browser.
